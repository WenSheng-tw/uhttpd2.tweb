 Installation

This version of TWeb is only for Httpd2.

We have 2 libraries, one to link with harbour to our server and another one to use with the user's frontend.

Backend
Install the library to compile with harbour in lib/tweb
To compile, add the tweb.lib library
-Llib\tweb
-ltweb

Frontend
The library for the frontend can be installed in files/tweb
