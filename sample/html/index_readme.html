Readme !

 Short summary

Basically TWeb is a set of commands that parse the different tags, functions,... of html/js/css and that helps any programmer to make the leap to the web in an easier way.

The best way to learn how to use TWeb is by testing and experimenting with the different examples found in this repository.


 Code example

<?prg
#include "lib/tweb/tweb.ch" 

  LOCAL o, oWeb
  
  DEFINE WEB oWeb TITLE 'Init TWeb'
		
  DEFINE FORM o OF oWeb

  INIT FORM o
    SAY VALUE '<h1>Welcome to TWeb !<hr>' ALIGN 'center' GRID 12  OF o
  ENDFORM o	
	
  INIT WEB oWeb RETURN 	
?>
Run example
